**npm init** - cria o novo pacote nodejs ex: npm init
- confirmar nome (nome pasta)
- confirmar versao
- descrição do pacote
- nome arquivo
-....

**npm install** - instala pacote ex: npm install nome_pacote

**npm publish** - publica 

**index.js** - arquivo

**npm rm** - desinstalar pacote ex: npm rm nome_pacote
